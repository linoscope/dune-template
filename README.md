# dune-template
A project template for OCaml + dune

# Commands
## Build
```
dune build
```

## Execute
```
dune exec ./bin/main.exe
```

## Test
```
dune test
```

# How to use
1. Clone the project
  ```sh
  git clone git@github.com:linoscope/dune-template.git
  ```
2.  Replace any occurance of `dunetemplate` with your own project's name
